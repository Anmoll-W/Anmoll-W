## Work That Shipped

Open source. You can read the code and the thinking.

### [ChalotripBot](https://github.com/Anmoll-W/chalotripbot)

`Next.js` `grammY` `Claude API` `Supabase` `Vercel`

A Telegram bot that handles the full group trip lifecycle — private budget collection, AI itinerary generation, bill splitting, and debt settlement inside one chat. Built alone, end to end.

**Problem:** Group trips collapse in chat chaos. Budgets, itineraries, and bill splits handled manually.

**Why it matters:** First AI product I designed, spec'd, and shipped solo.
